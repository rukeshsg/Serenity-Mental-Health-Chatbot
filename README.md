## 📊 Project Overview
Serenity is an AI-powered mental health support chatbot that provides safe, empathetic, and real-time conversations. It helps users manage stress, reflect on emotions, and access support in a private environment.

The system integrates a **BiLSTM-based intent model**, crisis detection, and safety mechanisms to ensure responsible and supportive interactions. It also features a modern UI for real-time communication and a personalized user experience.

---

## ✨ Features
- 💬 Real-time conversational support  
- 🧠 Intent detection (BiLSTM-based)  
- ⚠️ Crisis detection with safety responses  
- 🔍 RAG-based contextual replies  
- 📊 User insights and activity tracking  
- 🎨 Clean, calming UI  

---

## 🛠️ Tech Stack
- **Frontend:** Vite + React  
- **Backend:** Flask (Python)  
- **ML Model:** BiLSTM (Intent Classification)  
- **Database:** SQLite  
- **Search:** FAISS (RAG system)  

---

## 🚀 Purpose
Serenity aims to create a safe digital space where users can express emotions, receive guidance, and access support anytime.
